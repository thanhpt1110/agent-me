# agent-me Kickoff Discussion — 2026-05-09

_Originally scaffolded as "super-agent"; renamed later in this session to **agent-me** ("myself, but in agent mode" — public-shareable framework name)._

**Owner:** thaphan@nvidia.com
**Goal:** Một "Personal AI OS" tổng quản, 24/7, tự động hóa workload daily/weekly, lấy cảm hứng từ NVIDIA Personal Assistant.

## Yêu cầu gốc của user

- Super tool / agent quản lý + automate workload từ đầu đến cuối
- Dùng được hằng ngày + hằng tuần
- **Luôn online 24/7**
- **Connect GitHub** để lưu setup (config, prompts, skills) → source of truth
- Chạy trên **cloud agents** hoặc **server online có sẵn** của user
- Lấy cảm hứng từ Personal Assistant trong NVInfo
- Luôn dùng model tốt nhất + thinking tốt nhất → mặc định Claude Opus 4.7 (1M context)
- Không giới hạn tiền/server/thời gian → chọn cách tốt nhất

## Build blocks đã có sẵn (tận dụng được ngay)

- **MCP servers**: Jira, GitLab, Confluence, Glean, NVBugs, Jama, GDrive, OneDrive,
  SharePoint, MySQL, IPPSEC, Playwright, NSight-CUDA, Pagerduty
- **Daily-brief project** ở `~/daily-brief/` → integrate thành sub-agent đầu tiên
- **Skills available**: aiq-research, brev-cli, nvinfo-cli, loop, claude-api, init, review, security-review
- **Claude Code headless mode**: `claude -p "prompt"` cho cron jobs
- **CronCreate** trong Claude Code để self-schedule

## Roadmap đề xuất (sau khi chốt)

- **Phase 1 (1-2 ngày):** Tạo GitHub repo, scaffold Orchestrator + 1 sub-agent (sáp nhập daily-brief)
- **Phase 2 (3-5 ngày):** Deploy lên runtime 24/7 + setup cron weekly/daily
- **Phase 3 (1 tuần):** Thêm interface layer (Slack/Telegram bot)
- **Phase 4 (ongoing):** Mở rộng sub-agents, skills, automation theo nhu cầu
