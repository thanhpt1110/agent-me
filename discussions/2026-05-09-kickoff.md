# agent-me Kickoff Discussion — 2026-05-09

_Originally scaffolded as "super-agent"; renamed later in this session to **agent-me** ("myself, but in agent mode" — public-shareable framework name)._

**Owner:** thaphan@nvidia.com
**Goal:** A central "Personal AI OS" that runs 24/7, automates daily/weekly workload, and is inspired by NVIDIA Personal Assistant.

## Original User Requirements

- A super-tool / management agent that handles and automates workload end to end.
- Useful every day and every week.
- **Always online 24/7.**
- **Connect GitHub** to store setup (configs, prompts, skills) as the source of truth.
- Run on **cloud agents** or the user's existing always-online server.
- Take inspiration from Personal Assistant in NVInfo.
- Always use the best model and strongest reasoning mode; initial default was Claude Opus 4.7 (1M context).
- No hard budget limit on money/server/time; choose the best path.
